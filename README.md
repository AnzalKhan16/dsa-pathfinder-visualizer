# DSA Pathfinder Visualizer

An interactive pathfinding and graph algorithm visualizer that demonstrates how popular algorithms traverse grids and find shortest paths using animated visual representations.

---

## Features

- Interactive grid system
- Animated algorithm execution
- Obstacle generation
- Start and end node customization
- Real-time path visualization
- Speed control for animations

---

## Implemented Algorithms

- Breadth First Search (BFS)
- Depth First Search (DFS)
- Dijkstra's Algorithm
- A* Search Algorithm

---

## Tech Stack

- React.js
- JavaScript
- HTML5
- CSS3
- Vite



---

## Installation

```bash
git clone https://github.com/AnzalKhan16/dsa-pathfinder-visualizer.git

cd dsa-pathfinder-visualizer

npm install

npm run dev
```
